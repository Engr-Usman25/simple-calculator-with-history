# 🧮 Simple Calculator with History Logging

This Python script functions as a basic calculator that performs addition, subtraction, multiplication, and division. It also logs each calculation to a file, allowing users to view or clear their calculation history.

### ✨ Features

- Basic arithmetic operations: `+`, `-`, `*`, `/`
- Handles division by zero
- Saves calculations to a text file (`history.txt`)
- Supports viewing and clearing history
- Command-line interface

### 📂 File Structure

- `calculator.py` — Main script
- `history.txt` — Auto-created to store calculation history

### 🧠 Technologies Used

- File I/O in Python
- Loops and conditionals
- Exception handling
- String formatting

### ⚙️ How to Use

```bash
python calculator.py
